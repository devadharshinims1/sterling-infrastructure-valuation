## Objective
To evaluate the financial performance and estimate the intrinsic value of Sterling Infrastructure using ratio analysis and an Excel-based valuation model.

## Methodology

### 🔹 Financial Analysis
- Profitability ratios (ROE, Net Margin)
- Liquidity ratios (Current Ratio, Quick Ratio)
- Efficiency ratios (Asset Turnover)

### 🔹 Valuation Approach
- Ratio-based valuation
- Trend analysis of financial performance
- Comparative evaluation

---

## Key Insights

- The company demonstrates consistent revenue growth and improving profitability.
- Return on Equity (ROE) indicates efficient use of shareholder capital.
- Strong operating margins reflect effective cost management.
- Financial stability is supported by balanced liquidity and low leverage.
